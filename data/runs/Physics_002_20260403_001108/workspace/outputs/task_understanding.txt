Task objective: reproduce a fidelity-estimation workflow for arbitrary-geometry random quantum circuit sampling (RCS), using the provided experimental bitstring samples and matched ideal amplitudes/probabilities, and ultimately support depth-scan/size-scan comparisons that test the paper’s claimed gap between experimental fidelity and classical approximability.

Available data in this workspace: (1) read-only inputs under data/results and data/amplitudes; the observed dataset is centered on N=40 verification circuits, with many depth folders such as N40_d10_XEB, N40_d12_XEB, N40_d14_XEB, etc., and multiple circuit instances r=1..50; (2) XEB count files (*_counts.json) containing small measured subsets of bitstrings with counts; (3) matching amplitude files (*_amplitudes.json) containing ideal complex amplitudes for the same or overlapping bitstrings, from which ideal probabilities can be computed as |amp|^2; (4) additional MB-related files in data/results/*_MB, including counts and ideal-bitstring references; and (5) four reference papers in related_work/paper_000.pdf through paper_003.pdf.

Main planned analysis steps: parse the directory structure into (N,d,r,protocol) records; normalize bitstring key formats; convert amplitudes to ideal probabilities; join measured counts with ideal probabilities; compute per-instance counts-weighted XEB fidelity estimates and statistical uncertainties; inspect whether MB-regression-style analysis is also supported by the MB files; aggregate results across circuit instances at fixed N and varying d (and across other available settings if present); generate validation and comparison plots; and document assumptions, limitations, and consistency checks.

Expected deliverables: reproducible analysis code in code/; intermediate tables/results in outputs/ including per-(N,d,r) fidelity estimates with uncertainties; figures saved in report/images/ showing data overview and main fidelity/comparison curves; and a final report at report/report.md describing methodology, results, interpretation, and how the reproduced trends relate to the paper’s core conclusion.
